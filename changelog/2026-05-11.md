## Beta Watch: v2026.5.10-beta.2

Major beta with 100+ changes. Key items relevant to us:

### 🆕 New Features We Could Use

1. **`talk.realtime.instructions`** (#79081) — Append custom voice style instructions. If we ever do Discord voice, this is the knob.
2. **Private skill archive upload** (`skills.install.allowUploadedArchives`, #74430) — Install skills from zip archives. Could streamline custom skill distribution.
3. **Feishu topic-scoped threading** (#74903) — `message(action="send")` auto-threads inside Feishu topics. Not relevant to us (Discord-only) but shows multi-platform threading maturing.
4. **`provider/*` wildcard in `agents.defaults.models`** (#79485) — Dynamic model discovery for provider pickers without exact allowlists. Useful if we add new providers.
5. **1M context for Claude Opus/Sonnet 4** (#66766) — Reports correct 1M session context even when local config says 200k. We're running Opus 4.6 — this means our context window display is accurate now.

### 🔧 Fixes for Pain Points We've Hit

6. **Cron isolated self-cleanup can read own history** (#80019) — Cron jobs can inspect their own run history. Relevant to our cron-heavy setup.
7. **Config set persists values matching defaults** (#79856) — `config set`/`config patch` no longer silently drops values that happen to equal defaults. We've probably been bitten by this.
8. **Gateway config re-read after SIGUSR1 restart** (#79947) — Prevents stale config after restarts. We restart frequently during upgrades.
9. **Codex cross-fill `thread.id`/`thread.sessionId`** (#80137) — **Our own fix!** 🎉 Contributed by @kagura-agent. Fixes Codex app-server responses missing sessionId.
10. **Strip reasoning fields from replayed history** (#46637) — Prevents Qwen follow-up turns from rejecting stale `reasoning` payloads. We use Qwen3-32B in ABTI testing.

### ⚡ Worth Watching (Not Adopting Yet)

11. **Discord voice improvements** — Realtime diagnostics, barge-in detection, silence grace tuning. Comprehensive voice overhaul.
12. **QA/Mantis automation** — Telegram live PR evidence automation. Internal tooling signal.
13. **OpenRouter model fixes** — Multiple fixes for nested model IDs, budget limits, Perplexity Sonar. Important if we expand to OpenRouter.
14. **Agents/compaction exec session preservation** (#79284) — Scoped exec sessions survive compaction. Could fix subtle bugs in long-running cron jobs.
15. **LLM idle watchdog during stream setup** — Catches silent pre-stream model hangs. Safety net for our cron jobs.
