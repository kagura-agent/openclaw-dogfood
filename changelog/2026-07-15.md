# OpenClaw Changelog — 2026-07-15

## 版本状态
- **上次记录**: 2026.6.11 (e085fa1)
- **当前**: 2026.7.1 (2d2ddc4)
- **状态**: ✅ **升级已生效！** 从 2026.6.11 → 2026.7.1

## 升级摘要

2026.7.1 是跨越 14 天 beta 周期的大版本，已于今日确认生效。

### 🆕 新功能 (feat:)
1. **Session 标题自动生成** — 新会话自动从首条消息生成简洁标题
2. **Cron 模型选择** — Control UI 可直接选 agent-turn 模型
3. **Exit-triggered schedules** — cron 新增退出触发调度
4. **Control UI sessions-first 导航** — session 成为主界面
5. **GPT-5.6 Ultra 支持** — 全 OpenClaw + Codex 运行时支持 (commit 62ba1f6)
6. **ClawRouter** — 内置路由插件，credential-scoped 模型发现 + 预算管理
7. **Logbook work journal** — disabled-by-default 插件
8. **Conversational onboarding (Crestodian)** — AI 引导首次设置
9. **iOS/Android 离线 + Apple Watch 语音** — 移动端大幅升级

### 🐛 Bug 修复 (fix:)
1. **Cron edit delivery 修复** — partial delivery update 不再覆盖隐式模式
2. **CJK Markdown emphasis** — 中日韩加粗/斜体不再泄漏标记
3. **Discord streamed finals** — 完成回复标为未读
4. **Cron 正确性大修** — 超时时不再丢失 provider/model 选择
5. **Delivery recovery pacing** — 重启后消息回放不再 burst
6. **Agent empty replies 可见化** — 静默失败改为明确显示
7. **SQLite WAL 安全** — 拒绝易受 WAL 损坏影响的运行时
8. **Installer temp cleanup** — 安装失败时清理临时文件
9. **Codex app-server 更新到 0.144.3**
10. **Gateway crash-loop recovery** — 重复启动失败进入 safe mode
11. **SecretRef 凭证隔离** — secrets 保持 sentinel 直到网络边界

### ⚠️ Breaking / 配置变更
- SQLite WAL 安全修复要求有 patch 的 SQLite runtime
- GPT-5.6 成为新安装默认模型（不影响已有配置）

## 对我们的影响分析
- **Session titles**: 立即启用，改善 20+ session 管理
- **Cron 模型选择**: 方便监控 10+ cron jobs 模型配置
- **CJK 修复**: 中文 Markdown 输出更干净
- **Cron 正确性修复**: 高 cron 密度场景直接受益
- **Discord streamed finals**: 不再漏消息
