## 2026.4.24 Highlights (Current — installed since last check)

This was a major release focused on Google Meet, DeepSeek V4, browser automation, and platform hardening.

### Key Changes We Benefit From

1. **Google Meet participant plugin** — Not directly relevant to us yet, but shows platform expanding into real-time meeting scenarios.
2. **DeepSeek V4 Flash/Pro in catalog** — V4 Flash as onboarding default. We use Claude Opus but could evaluate DeepSeek for cost-sensitive cron jobs.
3. **Talk/Voice Call realtime voice loops** — Can consult full OpenClaw agent for deeper answers. Not currently using voice.
4. **Browser automation improvements** — Coordinate clicks, longer budgets, per-profile headless overrides. Useful if we expand browser-based research.
5. **Static model catalogs, lazy provider deps** — Lighter startup. Good for our 15+ cron setup.

### Key Fixes That Help Us

1. **Heartbeat clamp for oversized delays** — Prevents 1ms crash loop from `every` values over Node's timeout cap. Safety net for our cron setup.
2. **Session transcript fork fix** — Falls back to transcript-estimated parent token counts when cached totals are stale. Prevents oversized thread forks from cloning full parent.
3. **MCP cleanup on session end** — Bundled MCP runtimes retired at end of one-shot runs. Prevents accumulated stdio child processes. Good for our 15+ cron jobs.
4. **Gateway restart continuation** — Durable restart continuations with session-delivery queue. Better recovery from restarts affecting our cron fleet.
5. **Tool-result pruning hardening** — Malformed `{ type: "text" }` blocks can no longer bypass trimming. Stability improvement.
6. **Nix Home Manager PATH fix** — Relevant if we ever move to Nix.
