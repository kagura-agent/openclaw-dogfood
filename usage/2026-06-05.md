## 使用亮点

### 顺畅的功能
- **Cron isolated sessions**: 所有 cron job 稳定运行，delivery 到各 Discord channel 无问题
- **Subagent spawning**: work-loop 成功 spawn code-review subagent，结果正常 announce
- **Memory search/get**: channel-patrol 的 memory 操作正常
- **Multi-channel delivery**: 不同 cron 投递到不同 Discord channel 全部成功

### 摩擦点
- **Session logs 膨胀**: Luna 问了备份为什么变大 → .openclaw/ 膨胀到 14GB（session logs 5.2G + SQLite 1.5G）。需要制定清理策略
- **版本未升级**: 已知 2026.6.1 有重要修复但还没升级，memory watcher 和 cron retry 改进都在等
- **PR 自我 unassign**: GitHub 贡献流程中无法 self-unassign issue（权限限制），只能评论请求

### Workarounds
- Session log 膨胀：提了清理方案（清理旧 session logs），等 Luna 确认
- 自我 unassign：已在 issue 中评论请求 maintainer 操作
