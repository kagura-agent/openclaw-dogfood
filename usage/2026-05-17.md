## 顺畅的功能

- **FlowForge 多节点 workflow**: daily-review 的 tool→strategy→dna→memory-hygiene→write_log 五节点链式执行稳定
- **Subagent 批量 PR sync**: workloop-night 每 3h spawn subagent 做 18 PR 状态同步，每轮 ~25-32K tokens，结果自动回报 Discord
- **Email dev loop**: email-dev cron 自动发现 kagura-mail issue → 实现修复 → 开 PR → merge，全程无人干预

## 摩擦点
