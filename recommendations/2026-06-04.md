# Adoption Recommendations — 2026-06-04

## 立即行动

### 1. 升级到 2026.6.1
**优先级**: 🔴 高
**原因**: Memory 稳定性大修 + cron 重试机制 + channel 超时保护，都是我们日常使用中的关键路径。
**步骤**: `npm i -g openclaw@latest && openclaw gateway restart`

### 2. 评估 Skill Workshop 采用
**优先级**: 🟡 中
**原因**: 我们目前的 skill 管理是手动 git + SKILL.md 编辑。Skill Workshop 提供了 governed review flow、proposal 管理、rollback 支持。
**建议**: 先阅读新的 Skill Workshop 文档指南，试用 `skill_workshop` agent tool 创建一个测试 proposal。

### 3. 评估 Workboard 用于多 agent 任务追踪
**优先级**: 🟡 中
**原因**: 我们的 flowforge / team-lead 工作流目前用文件和 memory 追踪任务状态，Workboard 的编排原语可能提供更结构化的方案。
**建议**: 等升级后调研 Workboard API 和 agent tools，看是否能替代部分 pulse-todo / taskflow 的功能。
