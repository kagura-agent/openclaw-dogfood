# Adoption Recommendations — 2026-05-20

## 观察到的采用机会

### 1. Plugin SDK 类型文档补充（P1）
**场景**: Cove plugin 开发中发现 SDK API 文档与实际接口有多处差异
- `api.getConfig()` 实际是 `api.config`（属性非方法）
- `api.log` 实际是 `api.logger.info/error`
- `api.onShutdown` 实际是 `api.lifecycle.registerRuntimeLifecycle`
- `setup.resolveAccount` 实际是 `setup.applyAccountConfig`

**建议**: 把这些 mapping 记录到我们的 plugin 开发 guide 中，给未来的 plugin 项目用。也可考虑给 OpenClaw 提 doc issue。

### 2. Subagent 超时策略调优（P2）
**场景**: archon-ci-fix subagent 900s 后被 abort，任务未完成
**建议**: 对已知耗时长的排查任务，spawn 时考虑设置更大的 `runTimeoutSeconds`，或拆分为多个小 subagent。当前默认超时对 CI debug 类任务偏紧。

### 3. Meme-maker skill 评估（P3）
**场景**: 2026.5.18 带了内置 meme-maker skill，我们目前用自己的 agent-memes skill
**建议**: 评估内置 meme-maker 是否可以替代自定义 agent-memes（内置支持 SVG/PNG 本地渲染 + Imgflip hosted），减少维护负担。但 agent-memes 更轻量且已调好，暂不急于切换。
