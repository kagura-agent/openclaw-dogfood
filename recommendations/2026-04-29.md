# Adoption Recommendations — 2026-04-29

## Priority: High

### 1. Configure `dreaming.model` for cheaper dream sessions
**Feature**: New `dreaming.model` config knob (2026.4.26)
**Why**: We run 12 dream sessions nightly on Opus 4.6. Creative writing doesn't need Opus-level reasoning. Sonnet 4.6 or even a cheaper model would work fine.
**Savings**: ~50-70% cost reduction on dreaming, which is a significant portion of nightly token spend.
**How**: Add `dreaming.model: "claude-sonnet-4-20250514"` to config.

### 2. Set `tools.exec.timeoutSec` globally
**Feature**: Background/yieldMs exec commands now respect configured timeout (2026.4.26)
**Why**: We've had runaway background processes. This was literally our bug report (#67600). Now we can set a sane default.
**How**: `config.patch` with `tools.exec.timeoutSec: 300` (5 min default).

### 3. Fix web_search provider

**How**: Either restore MiniMax key or switch to Brave API (already configured per TOOLS.md).
