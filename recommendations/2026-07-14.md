# 采用建议 — 2026-07-14

## 🔴 建议立即采用（升级后）

### 1. 升级到 2026.7.1
- **优先级**: 高
- **理由**: 大版本，session titles + cron model 可见性 + CJK 修复 + Discord 未读修复，全是我们的痛点
- **行动**: `npm i -g openclaw@2026.7.1` + `openclaw gateway restart`
