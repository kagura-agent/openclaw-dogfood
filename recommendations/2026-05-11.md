# Adoption Recommendations - 2026-05-11

## From v2026.5.10-beta.2 (When Stable)

### 1. Strip Reasoning Fields for Qwen (`compat.stripReasoningReplay`)
- **What**: Automatically strips stale `reasoning` fields from replayed history for OpenAI-compatible providers
- **Why**: We test Qwen3-32B in ABTI. Follow-up turns can stall on old reasoning payloads. This fix (#46637) prevents that.
- **When**: After v2026.5.10 goes stable
- **Action**: No config needed — it's default behavior. Just upgrade.

### 2. LLM Idle Watchdog During Stream Setup
- **What**: Watchdog now catches silent pre-stream model hangs
- **Why**: Our cron jobs sometimes hang on model connection. This catches hangs earlier, before the full agent timeout.
- **When**: After upgrade
- **Action**: Automatic. May want to tune timeout if we see false positives.

### 3. Config Persistence Fix (#79856)
- **What**: `config set`/`config patch` now persist values that equal runtime defaults
- **Why**: We've likely had config changes silently dropped. After upgrade, re-verify any config that seemed to "not stick."
- **Action**: After upgrade, audit config values we've set that match defaults.

### 4. Cron Self-Inspection (#80019)
- **What**: Isolated cron runs can now inspect their own job run history
- **Why**: Useful for cron jobs that need to check if they already ran or need to reference prior run results.
- **Action**: Consider updating daily-audit cron to check its own prior run results for consistency.
