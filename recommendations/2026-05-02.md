# Adoption Recommendations — 2026-05-02

## Recommended for Immediate Adoption

### 1. Upgrade to v2026.4.29
Two releases behind. Key improvements for us:
- Discord group chat privacy defaults (our own bug report!)
- Cron timeout stuck-session fixes
- Duplicate Discord gateway monitor suppression
- Better Active Memory partial recall on timeout

### 2. Enable `commitments.enabled`
```json
{
  "commitments": {
    "enabled": true,
    "maxPerDay": 10
  }
}
```
Tracks inferred follow-up commitments from conversations and delivers reminders via heartbeat. Would complement our pulse-todo skill.

### 3. Active Memory chat-scoped filters
```json
{
  "memory": {
    "activeMemory": {
      "allowedChatIds": ["channel:xxx", "channel:yyy"]
    }
  }
}
```
Focus recall on important channels (evolution, infra) and skip noisy ones.

### 4. `messages.visibleReplies` for group chats
Already partially in effect with the new private-by-default group behavior, but explicitly configuring gives us control.

## Worth Exploring Later

### 5. People Wiki
Memory wiki with person cards could help with our community interaction tracking. Wait for it to stabilize.

### 6. Gateway Startup Diagnostics
Enable `gateway.diagnostics.startup` if we ever need to debug slow boots.

### 7. `heartbeat.skipWhenBusy`
Prevents heartbeat/cron overlap on our single server. Worth enabling if we see contention.
