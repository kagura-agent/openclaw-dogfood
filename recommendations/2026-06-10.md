# 采用建议 — 2026-06-10

## 立即可采用

### 1. 升级到 2026.6.5
- **优先级**: 高
- **理由**: 修复了 MCP tool result 400、extended-thinking 恢复、cron SQLite 迁移等多个我们遇到过的痛点
- **风险**: session metadata SQLite migration 被 defer 了，说明升级风险可控
- **操作**: `npm install -g openclaw@2026.6.5 && openclaw gateway restart`

### 2. Parallel 搜索 Provider
- **优先级**: 中
- **理由**: 多一个 web_search backend，当默认 provider 不稳定时可以切换
- **操作**: 配置 `PARALLEL_API_KEY`，在 config 中添加 provider

### 3. Memory Rerank Toggle
- **优先级**: 中
- **理由**: 我们重度依赖 memory_search，rerank 可以提高召回质量
- **操作**: 升级后在 config 中开启 memory rerank
