# 采用建议 — 2026-05-21

## 建议采用

### 1. `defineToolPlugin` — 自定义 Tool Plugin SDK
**场景**: finance/trading 的 `paper_trade` 命令目前通过 exec CLI 调用，可以封装成 typed tool plugin
**收益**: 更好的类型安全、自动 schema 生成、tool 权限控制
**优先级**: 中 — 等升级到 2026.5.19 后评估
**行动**: 先用 `openclaw plugins init` 生成模板试试

### 2. Skills `--global` 管理
**场景**: meme-maker、github 等通用技能可以装为 global shared
**收益**: 减少重复安装，统一更新
**优先级**: 低 — 目前单 agent 不紧迫
**行动**: 升级后测试 `openclaw skills install --global`

### 3. Browser dialog handling
**场景**: browser-automation 遇到网站 cookie consent / alert 弹窗
**收益**: 不再需要手动 workaround
**优先级**: 中 — 下次 browser task 时试用
**行动**: 更新 browser-automation skill 文档

### 4. Config reload metadata
**场景**: 修改 config 后不确定是否需要重启
**收益**: 减少不必要重启
**优先级**: 低 — 升级后自动可用
**行动**: 无需主动操作

## 暂不采用
- Android realtime voice — 不用移动端
- Telegram topic isolation — 不用 Telegram
- QA-Lab — 我们不跑 release QA
