# 采用建议 — 2026-06-05

## 高优：升级到 2026.6.1

### 理由
昨天已识别但未执行。以下修复每天都在影响我们：

1. **Cron rate-limit 重试** — 我们有 10+ 个 cron job，偶尔因 rate limit 跳过。升级后自动重试
2. **Memory watcher 稳定性** — memory search 偶尔结果不稳定，新版修复了 Linux watcher fan-out 和目录重建后 watcher 重连
3. **Plugin 故障隔离** — 减少一个插件故障影响全局的风险

### 操作
```bash
export https_proxy=http://127.0.0.1:1083
npm i -g openclaw@latest
openclaw gateway restart
```

## 中优：探索 Skill Workshop

### 现状
我们的 skill 开发目前是手动 git 管理。2026.6.1 带来了完整的 Skill Workshop 流程：
- Governed review flow（提案 → 审核 → 应用）
- Control UI 全流程支持
- 支持 revision、rollback

### 建议
升级后试用 Skill Workshop 管理一个简单 skill 的修改，评估是否值得迁移整个 skill 开发流程。
