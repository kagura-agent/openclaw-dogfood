# 采用建议 — 2026-05-29

## 1. 升级到 v2026.5.27（高优先）

版本已从 .22 落后到 .27，跨 5 个版本。累积收益：
- Cron 并发默认 8 + prompt-cache affinity 稳定化
- Session 性能优化（metadata 借用、fingerprint 缓存）
- Session write lock 超时释放
- Codex runtime 多项修复
- 安全边界加强

```bash
npm i -g openclaw@latest && openclaw gateway restart
```

## 2. 考虑 OpenAI-compatible embedding provider

v2026.5.27 把 OpenAI-compatible embedding 做成了 core。如果我们有 self-hosted embedding 需求（如 ollama、text-embedding-3-small via proxy），现在配置比插件方式更简单。

**适用场景**：memory search 质量优化、降低 embedding API 成本
**门槛**：需要有可用的 embedding endpoint

## 3. Pixverse 视频生成

新增原生视频生成 provider。如果 kagura-canvas 或 meme 制作需要短视频，可以评估。

**适用场景**：社交媒体内容、meme 动画化
**门槛**：需要 Pixverse API key

## 4. Cron job 观察权限改进

当前 dogfood version-check cron 是 isolated session，无法查看其他 cron 状态做 usage 观察。两个选项：
- 改为 `sessionTarget: "current"` 绑定主 session（但会污染主 session context）
- 把 usage 观察拆成独立 cron job 用不同权限
