# 采用建议 2026-05-28

## 立即采用

### 1. Reaction 审批
- **功能**: Signal/iMessage/WhatsApp 支持竖大拇指 reaction 来 approve exec
- **场景**: 手机端收到 elevated exec 请求时，直接 react 👍 即可
- **行动**: 升级到 v2026.5.26 后自动生效，无需额外配置

### 2. Cron `+5m` 相对时间
- **功能**: 一次性 cron job 支持 `--at +5m` 相对时间
- **场景**: 快速设置 "5分钟后提醒我" 类型的 reminder
- **行动**: 在 agent prompt 里告知可以用 `+5m` 格式

### 3. Activity Tab 调试
- **功能**: Control UI Activity 标签页实时显示 tool 活动
- **场景**: 调试 cron isolated run 时不用看日志，直接看 Activity
- **行动**: 打开 Control UI → Activity tab

## 下次迭代考虑

### 4. OpenTelemetry 可观测性
- **功能**: LLM content spans, alertable signals (blocked tools, failover, stale sessions)
- **场景**: 当我们的 cron 系统变复杂后，需要结构化 tracing
- **前置**: 需要部署 OTLP collector

### 5. QA coverage --match
- **功能**: 按 query 筛选 proof scenarios
- **场景**: 自动化测试特定 workflow 是否正常
- **前置**: 需要先建立 QA scenario metadata
