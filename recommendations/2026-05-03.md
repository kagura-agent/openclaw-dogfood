# Adoption Recommendations — 2026-05-03

## Priority: Upgrade to v2026.5.2

We are 4 releases behind. v2026.5.2 contains critical stability fixes that directly affect our setup:

### 🔴 Upgrade ASAP

1. **Heartbeat timezone fix** (#75487) — Our `activeHours.timezone: Asia/Shanghai` hasn't been correctly influencing timer scheduling. This explains occasional off-hours heartbeat activity.

2. **Heartbeat cooldown for exec-event floods** (#64016) — Runaway heartbeat issue where 30m intervals fire every 10s. We haven't seen extreme cases but this is a ticking time bomb.

3. **Isolated cron announce noise** (#61426) — Cron deliveries accumulating in main session. We've noticed main session growing; this is likely the cause.

4. **Session stuck state** — Sessions staying `running` after completion. We've hit this occasionally.

### 🟡 Adopt After Upgrade

5. **`heartbeat_respond` tool** — Replace `HEARTBEAT_OK` string returns with structured responses. Cleaner heartbeat outcomes.
   - Config: Available by default after upgrade, no config needed.

6. **`threadBindings.spawnSessions`** — Run `openclaw doctor --fix` to migrate legacy thread-spawn toggles.
   - Action: `openclaw doctor --fix` post-upgrade.

7. **Discord access groups** — Define reusable access groups for our Discord allowlists.
   - Config: Add `accessGroup:core-team` entries to channel config.

8. **`openclaw proxy validate`** — Add to toolchain-health cron.
   - Action: `openclaw proxy validate` in health checks.

### 🟢 Evaluate Later

9. **`compaction.midTurnPrecheck`** — Monitor if long tool loops cause context pressure, then enable.
   - Config: `agents.defaults.compaction.midTurnPrecheck: true`

10. **`skipOptionalBootstrapFiles`** — Audit which bootstrap files are noise and skip them.
